Sobre o projeto

O OilTest é um sistema acadêmico desenvolvido no Senac PR para demonstrar como sensores de baixo custo (Arduino) podem ajudar indústrias a reduzir o desperdício de óleos industriais.


O sistema coleta dados reais de turbidez, temperatura, cor e densidade do fluido, e classifica automaticamente cada amostra em três categorias: pronto para uso, precisa de filtragem ou deve ser descartado.


Orientação: Prof. Anselmo Rocha · Equipe: Luan Lourenço, Miguel Valenga, Eduardo Oliveira, Gustavo Cordeiro, João Vitor Portes, Octavio Henrique.
